# Student Management System (C++)

A simple console-based Student Management System built using C++. This program allows users to manage student records through a menu-driven interface.

## Features

- Add new student records
- Display all student records
- Search student by roll number
- Delete student record
- Simple menu-driven console interface

## Technologies Used

- C++
- Structures
- Arrays
- Functions
- Loops and Conditional Statements
